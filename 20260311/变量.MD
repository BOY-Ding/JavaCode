# 基本类型 
## 六种数字类型 byte short int long float long
## 字符 char 2个字节
### char实际是数字，例如byte n = 'a',存的是对应的ascII码97
### 例如char b = (int)n + 1, 输出的结果是字母b
## Boolean 1个字节

#基本类型可以自动转换
## char - int - long - float - double
## byte - short - int - long - float - double
### char不能转换成byte和short， 三者在运行时自动升级成int
### 在声明byte/short时，如果值符合范围则可以使用，不会校验，运行时才会校验
### 运行时，所有会自动设置为最大范围的类型
